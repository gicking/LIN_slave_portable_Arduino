◆ handler()

void LIN_Slave_SoftwareSerial::handler ( void  )
virtual

Handle LIN protocol and call user-defined frame handlers.

Handle LIN protocol and call user-defined frame handlers, both for master request and slave response frames. BREAK detection is based on inter-frame timing only (Arduino doesn't store framing error) -> less reliable. Notes:

  • received BREAK byte is consumed here to support also sync on SYNC byte
  • ESP32 & ESP8266 SoftwareSerial ignores bytes w/o stop bity -> use SYNC(=0x55) for frame synchronization
